(Note that a negative number means shifting to the right)

6LastSNSN [(BhhhB + PppPpPP - SerialsSerialSN % LitInd) + 1/3] - 7 % 3

  • B = Batteries
  • h = Battery holders
  • Bh = Batteries plus holders
  • hB = Highest number in SN
  • P = Ports

  • SN = Module count
